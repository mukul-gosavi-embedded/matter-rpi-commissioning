# Matter over BLE/Wi-Fi on Raspberry Pi using Yocto Linux

## Objective

Build a custom embedded Linux Matter platform using:
- Raspberry Pi 5 as Matter Commissioner
- Raspberry Pi 3 as Matter Accessory
- Yocto Linux
- BLE commissioning
- Wi-Fi operational transport
- Future kernel-space GPIO integration

## Current Phase

Phase 1 — Yocto platform bring-up and repository foundation.

- Yocto Linux
